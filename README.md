# End_to_End_Medical_Chatbot
End to End Medical Chatbot Project Implementation
# How to Run?
### Steps:
Clone the Repository
Project repo: https://github.com/KumodhiRajapura/End_to_End_Medical_Chatbot.git

### Step 01- Create a conda environment after oprning the repository

```bash
conda create -n medibot python=3.10 -y
```

```bash
conda activate medibot
```
### Step 02- Install the Requirements

```bash
pip install -r requirements.txt
```
